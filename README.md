# Typewriter solver

## English
### About
This is a small chrome extension that solves exercises on [Typewriter](https://at4.typewriter.at/) for you.
It's completly opensource and doesn't depend on any libraries, except of Chrome API and JavaScript itself.

### Features
- Randomized speed
- Intentional mi**s**stakes
- Settings you can configure
